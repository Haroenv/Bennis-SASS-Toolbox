    Description

    Mixin to use in @media elements to hide an element but make it accessible to screenreaders

    Parameters

    None.

    Example

    Usage

   overflow: hidden;

}

    clearfix

    @mixin clearfix() { ... }

    Description

    Clearfix - For modern browsers

    1. The space content is one way to avoid an Opera bug when the contenteditable attribute is included anywhere else in the document. Otherwise it causes space to appear at the top and bottom of elements that are clearfixed.
    2. The use of table rather than block is only necessary if using :before to contain the top-margins of child elements.

    Parameters

    None.

    Example

    Usage

    .SomeClass {
  @include clearfix();
}

    CSS output

    .SomeClass::before,
.SomeClass::after {
    content: " "; // 1
    display: table; // 2
}
.SomeClass::after {
    clear: both;
}

    Used by

    • [placeholder] clearfix

    Links

    • http://nicolasgallagher.com/micro-clearfix-hack/

    Author

    • Nicolas Gallagher

    unitize

    @function unitize($list, $unit: px, $font-size: $unitless-font-size-base) { ... }

    Description

    Converts the unitless numerical values in a given SASS list to values in a given unit

    See the SASS_REFERENCE for a description and examples of the "list" data type: http://sass-lang.com/docs/yardoc/file.SASS_REFERENCE.html#lists

    Accepts optional $font-size param (defaulting to $unitless-font-size-base) for "em" conversions

    NOTES:

    CSS font shorthand (e.g., "bold 12/18 Helvetica, Arial, sans-serif") is not yet supported;

    see "Division and /": http://sass-lang.com/docs/yardoc/file.SASS_REFERENCE.html#division-and-slash

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $list

    Unitless Pixel value

    		String—none
    $unit

    Unit

    		Stringpx
    $font-size

    Defaults to value of $unitless-font-size-base

    		Number$unitless-font-size-base

    Example

    Usage

    See CSS rulesets below the function
